## ‚ú® Features
- Paste a macro in English, German, or French ‚Üí get the translated version instantly.
- Translates **actions, items, mounts, emotes, places, duties, statuses, achievements, titles, gearsets**, and more.

- Supports **full macro commands** as of patch **7.31 (Dawntrail)**.
- Offline dictionary (`dictionary.json`) generated with XIVAPI, no backend required.
- Responsive design: works on PC, tablet, and phone.
- Animated glowing theme inspired by Final Fantasy XIV UI.
